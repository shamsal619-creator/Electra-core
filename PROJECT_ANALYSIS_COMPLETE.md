## الملخص التنفيذي

ElectraCore هي منصة تجارة إلكترونية كاملة المكدس مبنية بتقنيات الويب الحديثة. يتكون النظام من:

- **الواجهة الأمامية**: جافاسكريبت عادي (ES6+)، HTML5، CSS3 - واجهة مستخدم متجاوبة وتفاعلية
- **الخلفية**: Node.js مع إطار عمل Express - خادم API
- **قاعدة البيانات**: MongoDB Atlas - قاعدة بيانات NoSQL مستضافة سحابياً
- **الأمان**: bcryptjs لتجزئة كلمات المرور، التحقق من الصحة، CORS مفعّل
- **البيئة**: التكوين مدار عبر متغيرات البيئة

## 3. قاعدة البيانات والطبقة البيانية

### **هيكل البيانات**

**مجموعة المستخدمين**:
```
_id: معرف MongoDB تلقائياً
first: الاسم الأول (2+ أحرف مطلوبة)
last: الاسم الأخير (2+ أحرف مطلوبة)
email: البريد الإلكتروني (فريد، محقق بـ regex)
password: كلمة المرور المجزأة (6+ أحرف)
nick: الاسم المستعار (اختياري)
gender: الجنس (اختياري)
language: اللغة المفضلة (اختياري)
country: الدولة (اختياري)
timezone: المنطقة الزمنية (اختياري)
address: عنوان التسليم (اختياري)
created: تاريخ الإنشاء
createdAt: طابع زمني تلقائي
updatedAt: طابع زمني تلقائي
```

**بيانات المنتجات**:
```
id: معرف فريد
category: الفئة (أجهزة كمبيوتر محمولة، هواتف، إلخ)
name: اسم المنتج
price: السعر الحالي
oldPrice: السعر الأصلي
image: رابط الصورة
description: الوصف
brand: العلامة التجارية
color: اللون
inStock: توفر المخزون
```

### **حيث يتم تخزين البيانات**

1. **MongoDB Atlas** - حسابات المستخدمين
2. **LocalStorage** - عناصر سلة التسوق
3. **مصفوفة JavaScript** - كتالوج المنتجات

### **كيفية إنشاء البيانات**

**تسجيل المستخدمين**:
1. المستخدم يملأ نموذج signup.html
2. إرسال POST `/api/register`
3. التحقق من الصحة والتقليص
4. فحص البريد الإلكتروني المكرر
5. تجزئة كلمة المرور
6. حفظ في MongoDB

**المنتجات**:
- مشفرة في `/public/js/products.js`
- لا توجد واجهة برمجية لبذر البيانات

**سلة التسوق**:
- محلي 100% في LocalStorage
- لا تتطلب خادم

### **الاتصال والاستعلامات**

**تسجيل الدخول**:
- POST `/api/login` {email, password}
- البحث عن المستخدم
- مقارنة كلمة المرور
- إرجاع بيانات المستخدم

**تحديث الملف الشخصي**:
- PUT `/api/profile` {userId, fields}
- تحديث وثيقة MongoDB
- إرجاع الملف الشخصي المحدث

**سلة التسوق**:
- بدون اتصال بالخادم
- جميع العمليات محلية
