## 🗄️ قاعدة البيانات

**Firebase Firestore** - قاعدة البيانات السحابية الرئيسية الوحيدة
- تخزين بيانات المستخدمين (اسم، بريد إلكتروني، كلمة المرور)
- تخزين بيانات المنتجات (اسم، سعر، وصف، صور)

**Firebase Storage** - التخزين السحابي
- تخزين صور المنتجات
